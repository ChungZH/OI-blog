# 题解

这题还是比较简单的，考的就是 **把十进制数字转成二进制数字** 以及 **判断数字是否回文**。

判断回文最简单的方法就是：把一个字符串翻转后，判断是否还等于原字符串。如果等于，就是回文。不等于就不是回文。

核心代码如下：

```cpp
// a 和 b 都是字符串，N 是输入的数

while (N)
{
    if (N % 2 == 0)
        a += '0';
    else
        a += '1';

    N /= 2;
}

b = a;
reverse(b.begin(), b.end());

// 接下来判断两个字符串是否相等，然后输出即可
```
