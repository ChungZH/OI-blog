## 题解

其实是一道很简单的题，个人感觉难度 `普及-`，就是一道前缀和题目。

不知道前缀和的话可以看这里 [前缀和 - OI Wiki](https://oi-wiki.org/basic/prefix-sum/)。

用 `b` 数组作为前缀和数组，然后算 `b` 中 **最大的长度为 k 的区间的和**。

```cpp
#include <iostream>
using namespace std;
int a[100001], b[100001];
int main()
{
    int n, k, maxx = 0;
    cin >> n >> k;
    
    for (int i = 1; i <= n; i++)
    {
        cin >> a[i];
    }
    
    b[1] = a[1];
    
    for (int i = 2; i <= n; i++)
    {
        b[i] = b[i-1] + a[i];
    }
    
    for (int i = k; i <= n; i++)
    {
        if (b[i] - b[i-k] > maxx) maxx = b[i] - b[i-k];
    }
    
    cout << maxx << endl;
    return 0;
}
```
